🐍 Snake Game — Desktop Application

Application desktop interactive du jeu Snake, développée avec React et Electron.

🚀 Stack Technique

Frontend : React 18 (Hooks, gestion d’état, gestion des événements)

Desktop : Electron 28 (Windows / macOS / Linux)

Styling : Tailwind CSS

Audio : Web Audio API (effets sonores dynamiques)

⚙️ Fonctionnalités

Architecture modulaire basée sur des composants React réutilisables

Logique complète du jeu : détection des collisions, système de score, gestion des déplacements

3 niveaux de difficulté avec vitesses dynamiques (70ms à 180ms)

Interface interactive : menus, overlays, feedback visuel et sonore

Optimisation des performances (game loop via useRef, gestion des effets et cleanup)

Configuration de build via Electron Builder

▶️ Installation
npm install
npm run electron-dev
📦 Build
npm run build
